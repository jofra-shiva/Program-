Complex Number Addition

Aim:

To perform addition of two complex numbers.

Algorithm:

  1. Define a class named "complex" with integer variables for real and imaginary parts.
  2. Define a default constructor and a parameterized constructor to initialize the complex numbers.
  3. Define a method "addcomplex" to add two complex numbers.
  4. In the main class "complexmain", create instances of the complex class and add two complex numbers.
  5. Display the result of the addition.

Program:

            
                import java.io.*;

                class complex {
                    int real, img;

                    complex() {}

                    complex(int Treal, int Timg) {
                        real = Treal;
                        img = Timg;
                    }

                    complex addcomplex(complex c1, complex c2) {
                        complex ctemp = new complex();
                        ctemp.real = c1.real + c2.real;
                        ctemp.img = c1.img + c2.img;
                        return ctemp;
                    }
                }

                class complexmain {
                    public static void main(String args[]) throws IOException {
                        complex c1 = new complex(4, 8);
                        complex c2 = new complex(5, 7);
                        complex c3 = new complex();
                        c3 = c3.addcomplex(c1, c2);
                        
                        // Output in red color
                        System.out.println("\u001B[31mReal value: " + c3.real);
                        System.out.println("Imag value: " + c3.img);
                        System.out.println("Complex addition: " + c3.real + "+i" + c3.img + "\u001B[0m"); // Reset color after output
                    }
                }
